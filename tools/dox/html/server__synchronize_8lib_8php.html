Finds the difference in source and target matching tables by first comparing source table's primary key entries with target table enteries.

It gets the field names for the matching table also for comparisons. If the entry is found in target table also then it is checked for the remaining field values also, in order to check whether update is required or not. If update is required, it is placed in $update_array Otherwise that entry is placed in the $insert_array.

Parameters:

array&$update_arrayA three dimensional array passed by reference to contain updates required for each matching table
array&$insert_arrayA three dimensional array passed by reference to contain inserts required for each matching table

Comparing each corresponding field of the source and target matching rows. Placing the primary key, value of primary key, field to be updated, and the new value of field to be updated in each row of the update array.

PMA_structureDiffInTables() Gets all the column information for source and target table.

Compare columns on their names. If column exists in target then compare Type, Null, Collation, Key, Default and Comment for that column. If column does not exist in target table then it is placed in $add_column_array. If column exists in target table but criteria is different then it is palced in $alter_str_array. If column does not exist in source table but is present in target table then it is placed in $uncommon_columns. Keys for all the source tables that have a corresponding target table are placed in $matching_tables_keys. Keys for all the target tables that have a corresponding source table are placed in $target_tables_keys.

Parameters:
string$src_dbname of source database
string$trg_dbname of target database
mixed$src_linkconnection established with source server
mixed$trg_linkconnection established with target server
array$matching_tablesnames of matching tables
array&$source_columnscolumns information of the source tables
array&$target_columnscolumns information of the target tables
array&$alter_str_arraythree dimensional associative array first index being the matching table index, second index being column name for which target column have some criteria different and third index containing the criteria which is different.
array&$add_column_arraytwo dimensional associative array, first index of the array contain the matching table number and second index contain the column name which is to be added in the target table
array&$uncommon_columnscolumns that are present in the target table but not in the source table
array$criteriacriteria which are to be checked for field that is present in source table and target table
array&$target_tables_keysfield names which is key in the target table
int$matching_table_indexnumber of the matching table
