PMA_wildcardEscapeForGrant ( $  dbname,
$  tablename 
)

Checks if a dropdown box has been used for selecting a database / table.

Escapes wildcard in a database+table specification before using it in a GRANT statement.

Escaping a wildcard character in a GRANT is only accepted at the global or database level, not at table level; this is why I remove the escaping character. Internally, in mysql.tables_priv.Db there are no escaping (for example test_db) but in mysql.db you'll see test for a db-specific privilege.

Parameters:
string$dbnameDatabase name
string$tablenameTable name
Returns:
string the escaped (if necessary) database.table
